# RatingBadge

Used in the hotel listing for guest ratings.

**Extends: [Box](/Box)**

```.jsx
<RatingBadge>
  9.0
</RatingBadge>
```

## Props

| Prop    | Type   | Description           |
| ------- | ------ | --------------------- |
| `color` | string | Sets text color       |
| `bg`    | string | Sets background color |
