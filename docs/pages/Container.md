# Container

Use the `Container` component to center content horizontally with a max-width.

```.jsx
<Container maxWidth={256} >
  <Box p={3} color='white' bg='blue'>
    Container
  </Box>
</Container>
```

## Props

| Prop       | Type   | Description                         |
| ---------- | ------ | ----------------------------------- |
| `maxWidth` | Number | Sets `max-width` (default `1280px`) |
