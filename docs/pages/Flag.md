# Flag

Use the `Flag` component for stylized merchandising messages.

```.jsx
<Card py={3}>
  <Flag>
    Hello
  </Flag>
</Card>
```

```.jsx
<Card py={3}>
  <Flag width={192} bg='purple'>
    Hello
  </Flag>
</Card>
```

## Props

| Prop    | Type   | Description      |
| ------- | ------ | ---------------- |
| `color` | string | Text color       |
| `bg`    | string | Background color |
